Rails API Concepts Based Learning
=================================

1. To Create Rails API Based Project
     
     $ rails new project_name --api   # default db = sqllite

2. To Create Rails API With Database
     $ rails new project_name --api --database=postgresql

3. To Create Rails API With VanilaJS
     $ rails new project_name --api --skip-coffee     # default is coffee script based js, --skip-coffee is VanilaJS

4. To Check For The Routes 
     $ rails routes                # to look for all the routes 
     $ rails routes | grep books   # using the grep linux fetch only the books related routes
